### 游戏中提及的事物

+ 船
+ 牧师
+ 魔鬼
+ 岸边

### 玩家动作表

| 对象 |   动作                |
|:------:|:----------------------------------:|
|牧师|上船（离岸）、下船（上岸）|
|魔鬼|上船（离岸）、下船（上岸）|
|船|两岸之间来回，只有船上有人才可|
